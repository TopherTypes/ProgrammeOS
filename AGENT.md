# AI Agent Context

## Purpose

This system is a programme leadership management platform supporting
programme managers and transformation leads.

The focus is leadership attention and governance tracking rather than
detailed project scheduling.

## Design Principles

1.  Table-first UI
2.  Modal workspace interaction
3.  Context-first creation
4.  Progressive disclosure forms
5.  Explicit relationship modelling

## Avoid

-   kanban boards
-   gantt charts
-   complex scheduling engines

This is not a full project management suite.

## Core Relationships

Project ↔ People (roles)

Meeting ↔ Context (project/person/programme)

Update / Decision / Action → Inform tracking

RAID → project or programme scope

## Development Guidance

Prefer: - clarity - readable code - explicit schemas

Avoid: - unnecessary abstraction - hidden behaviour

## Vision

The product should feel like:

Notion + governance tracker + meeting preparation tool.
